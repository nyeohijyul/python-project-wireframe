📐 화면 구성 & 데이터 흐름 설명

Step 1~4 — 질문 카드: 챗봇 대화 형태(버블 UI)로 질문을 제시하고, 카드 선택으로 답변합니다. 이전 답변이 오른쪽 버블로 누적되어 흐름을 보여줍니다.
Step 5 — AI 분석 로딩: ExerciseDB 필터링 → YouTube API 검색 → 상위 영상 선별 3단계를 진행 바로 시각화합니다. 선택 조건 칩을 보여줘 신뢰감을 줍니다.
Step 6 — 결과 화면: ExerciseDB 기반 동작 카드(근육군·장비·난이도 태그)와 각 동작별 YouTube 영상 3개를 가로 탭으로 탐색합니다. TOP PICK 뱃지로 최적 영상을 강조합니다.
YouTube 영상 목록: 동작 선택 탭 → 해당 동작 검색어로 YouTube API 호출 → 채널명·조회수·업로드일·좋아요 수 표시. "루틴에 추가" 버튼으로 바로 저장 가능합니다.
ExerciseDB 필터 조건: bodyPart(신체부위) + equipment(장비=bodyweight) + difficulty(난이도) + target(근육군)으로 4중 필터링합니다. 무릎 통증 선택 시 knee 관련 동작 자동 제외.
